# 💎 Jewelry Store (E-commerce Django Project)

## 📌 Description
This is an e-commerce web application built using Django.  
It allows users to browse jewelry products, view details, and manage product listings.

---

## 🚀 Features
- 🛍️ Product listing
- 🔍 Product details view
- 🖼️ Image upload for products
- 🧾 Category-wise products
- 🛒 Basic e-commerce functionality
- ⚙️ Admin panel for product management

---

## 🛠️ Tech Stack
- Python
- Django
- HTML, CSS
- SQLite

---
